# CPS510-EVL-Library-Database
CPS 510 Database Assignment

## Installation Instructions

For our application, we have a separate backend acting as an API and a frontend which uses the API to get and modify the database data. As such, the following commands and instructions are necessary to run and install the application.

**1. Clone and download on local machine:***
- git clone git@github.com:eleanortien/CPS510-EVL-Library-Database.git
- cd CPS510-EVL-Library-Database

**2. Connect to the universityâ€™s private network (use a VPN if needed)**
   
**3. Download backend dependencies and run API:**
- npm i
- npm run build
- npm start
    
**4. On a separate terminal, download frontend dependencies and run the web application**
- cd evl-library-front
- npm i
- npm start
The web application should now be accessible on http://localhost:4200.
Login with the same credentials as sql developer.
