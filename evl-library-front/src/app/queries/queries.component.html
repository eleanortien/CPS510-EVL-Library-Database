Simple Queries

Querie #1 Book Copies Available

This query will return all the books information (book_id, title, author, genre) and will count how many copies of that book are available to borrow.

Querie #2 Member Overdue Books

This query will return the member information (member_id, username) and the information for the book that is overdue (title, copy_id) and will count how many days overdue that book is.

Querie #3 Manager Information

This query will get the employee information (staff_id, first_name, last_name) and will cound all the different number of employees that they manage (part_time, volunteer, full_time) as well as the total number of emplyees under this manager.

Querie #4 Part-Time Staff With Above-Average Pay

This query will return the part-time staff information (staff_id, first_name, last_name) for employees who work at Branch 2. It also includes the branch name, and it only shows staff who earn more than the average hourly rate of all staff.

Querie #5 Volunteer With the Most Hours

This query will return the volunteer information (staff_id, first_name, last_name) for the volunteer who has volunteered the highest total number of hours.

Querie #6 Number of Branches per City in Ontario

This query will return each city in Ontario that has a branch and will count how many branches are located in that city.

Querie #7 Oldest Fantasy Book

This query will return the book information (title, author, publish date, genre) for the oldest published book in the Fantasy genre.

Querie #8 Members With Borrowed Books

This query will return the member information (member_id, username) along with the total number of books each member has borrowed.

Querie #9 Number of Unique Books per Branch

This query will return each branch (branch_id) and will count how many unique books (book_id) are located at that branch.

Querie #10 Books and Total Volumes per Supplier

This query will return supplier information (supplier_id) along with the number of distinct books each supplier has provided and the total number of physical copies supplied.

Querie #11 Next In Queue for Holds

This query will return queue information (member_id, book_id, place_in_queue) for all members who are currently in place 0 in queue and will be able to borrow the book next.

Querie #12 Suppliers With 10 or More Supplied Copies

This query will return supplier information (supplier_id, supplier_name, email, phone_number) for suppliers who have provided at least 10 total book copies.

Advance Queries

Querie #1 User Queue Positions

This query will return the username of the user 'lsu', and get all the hold information (book_title, queue_position) for all books that the user is currently waiting for.

Querie #2 Branch Overview

This query will return each branch's information (branch_id, branch_name, full_address) and will count how many staff members work there and how many distinct books are located at that branch.

Querie #3 Book Popularity

This query will return each book's information (book_id, title) and will calculate its popularity based on the number of borrows and queue holds it has.

Querie #4 Members With Borrows and Queue Holds

This query will return the usernames of members who have borrowed more than one book and also have at least one book in a queue hold.

Querie #5 Supplier Donation Percentage

This query will return supplier information (supplier_id, supplier_name) and will calculate what percentage of all book copies were provided by each supplier.

Querie #6 Books Only at Branch 2

This query will return the titles of books that are located at Branch 2 and are not located at any other branch.
